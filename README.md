# Todo React App

Приложение для управления списком дел, созданное с использованием React, TypeScript и Vite.

## Структура проекта

```
src/
├── types/
│   └── todo.ts          # Типы для Todo приложения
├── components/
│   ├── TodoForm.tsx     # Форма добавления задачи
│   ├── TodoList.tsx     # Список задач
│   └── TodoItem.tsx     # Элемент задачи
├── reducers/
│   └── todoReducer.ts   # Редьюсер для управления состоянием
├── App.tsx              # Главный компонент
├── main.tsx             # Точка входа
└── App.css              # Стили приложения
```

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev-сервер:
```bash
npm run dev
```

3. Соберите проект для production:
```bash
npm run build
```

## Функциональность

- Добавление новых задач
- Отметка задач как выполненных
- Удаление задач
- Редактирование задач (двойной клик или Enter после редактирования)
